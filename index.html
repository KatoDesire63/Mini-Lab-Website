Automated Laboratory Log Management System (ALMS)

Real-time monitoring of temperature, humidity, and pressure

Project Overview

This embedded system uses an ESP32 microcontroller connected to temperature, humidity, and pressure to monitor lab conditions in real time. Sensor data is transmitted to a user dashboard, enabling efficient and remote monitoring by laboratory personnel.

View GitHub Repo

Features

ESP32 Integration

Connects and communicates with sensors for real-time data collection and wireless transmission.

Environmental Sensing

Monitors temperature, humidity, and pressure for accurate lab condition tracking.

Real-Time Data

Instantly updates readings to ensure live insights and quick response to changes.

Dashboard Access

View all environmental data remotely from a centralized and user-friendly dashboard.
